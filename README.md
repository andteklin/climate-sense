# climate-sense
Arduino program integrating temperature and humidity sensors with smooth RGB-LED visuals and fan activation.

Authored by Andrew Lin, Antony Langley, Charlotte Lin, Sahil Shah, and Joshua Kim as part of EE Research at Stevens Institute of Technology in July 2022.
